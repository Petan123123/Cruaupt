# Cruaupt
- Cruaupt is a Luau cryptography library. Can be used in Roblox.

## Usage

> *There are two ways to use this library*

- Go to the 'source' folder and pick your assets there! OR:
- Download the rbxm file and import it to Roblox and use it as a single big module!
